# 🤖 Manvi Garg — Personal Portfolio Website

A modern, responsive personal portfolio website built with pure HTML, CSS, and Vanilla JavaScript — no frameworks, no build tools.

## ✨ Features

- Hero section with animated name reveal and gradient typography
- About section with profile card and stats
- Skills section categorized into AI/ML, AI Tools, Programming, and Workflow
- Projects showcase with tech stack tags
- Experience & Education timeline
- Certifications grid
- Contact section with form validation
- Dark / Light mode toggle
- Scroll-reveal animations
- Sticky navbar with smooth scrolling
- Fully responsive (mobile-first)

## 🛠️ Tech Stack

`HTML5` `CSS3` `Vanilla JavaScript` `Google Fonts`

## 🚀 Getting Started

1. Clone the repository
```bash
   git clone https://github.com/manvigarg11/portfolio.git
```
2. Add your resume as `Manvi_Resume(AI).pdf` in the root folder
3. Add your photo as `img.jpeg` in the root folder
4. Open `index.html` in your browser — done!
